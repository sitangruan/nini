nini is a lovable thing!

nini is a little REST service done with ASP.NET Core. Right now it is using .NET Core V2.1.

It is little but it is almighty! It contains these features:

1. Dependency Injection by Autofac
2. Swagger UI supported by Swashbuckle (with groupable end points)
3. Multi version supported
4. Logging with Nlog
5. Https supported with self signed certificate
6. XUnit test with Moq
7. User session management
8. Deployed as a self hosted Windows service.
